### How to use?

1. Clone the repo:
```sh 
$ git clone ...
```
2. Install the required modules:
```sh 
$ pip install -r reqs.txt
```
3. Make sure you have firefox
4. run the program: 
```sh 
$ python main.py
```
5. Enjoy!
